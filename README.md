# Music Audience Network Analysis:

### 🎵 Project Inspiration
[cite_start]This project was inspired by the global success of **ROSÉ & Bruno Mars's "APT."**[cite: 1, 23]. [cite_start]Beyond its viral nature, I wanted to visualize the **cross-cultural resonance** and the invisible "bridges" that connect disparate global fanbases[cite: 1, 23]. [cite_start]This tool is designed to help marketing teams to identify "low-resistance" migration paths for artists[cite: 1, 23].

### 🛠️ Technical Stack
* [cite_start]**Language**: Python [cite: 1, 23]
* [cite_start]**Data Source**: Last.fm API (Collaborative Filtering data) [cite: 1, 23]
* [cite_start]**Graph Theory**: NetworkX (Betweenness & Eigenvector Centrality) [cite: 1, 23]
* [cite_start]**Visualization**: Pyvis (Physics-based interactive rendering) [cite: 1, 23]
* [cite_start]**Analysis Tools**: Pandas, Requests [cite: 1, 20]

### 🧠 Core Methodology
1. [cite_start]**Automated ETL**: Built a recursive scraper to bypass streaming "black-box" limitations, extracting weighted audience overlap data[cite: 1, 20].
2. [cite_start]**Topological Metrics**: Calculated **Betweenness Centrality** to identify "Bridge Nodes"—artists like **ROSÉ** or **Bruno Mars** who serve as hubs between sub-cultures (e.g., K-Pop and Western Pop)[cite: 1, 23].
3. [cite_start]**Business Insight**: Defined "Bridge Potential" to help optimize **Marketing ROI** and lower **Customer Acquisition Costs (CAC)** by targeting high-influence micro-communities[cite: 1, 23].
